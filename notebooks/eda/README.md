# Exploratory Data Analysis notebooks

This folder contains the notebooks for Phase 1 (Data & Descriptive Statistics) of the NYC Urban Demand Intelligence Platform.

Order of execution:
1. 01_data_ingestion.ipynb – load raw NYC taxi data
2. 02_data_cleaning.ipynb – clean and validate raw trips
3. 03_feature_engineering.ipynb – build zone-hour features
4. 04_descriptive_stats.ipynb – summary statistics
5. 05_visualize_demand.ipynb – demand distribution and temporal/spatial patterns
6. 06_probability_fitting.ipynb – probabilistic modeling of demand
7. 07_insights_summary.ipynb – consolidated analytical and business insights
